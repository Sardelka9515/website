System time is more important than you think.
Incorrect system time may result in various connection troubles(invalid credential, expired access token, unknown certificates, etc..). Windows does provide a way to sync your system with Microsoft's time server, but doesn't seems to be working properly on ancient Windows(XP and below). Wrong BIOS time or broken CMOS battery can also be the reason.
This program allows you to sync system with a global, reliabe ntp time server. You can also set desired interval to sync automatically every few minutes.
