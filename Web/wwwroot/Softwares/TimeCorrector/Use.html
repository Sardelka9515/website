Run the program as administrator.
The first TextBox lets you set your ntp time server to sync. Leave it unchanged normally.
The second TextBox lets you customize the sync interval. Only a floating-point numeric value is allowed.
If you started the automatic sync procedure, you will see a TextBlock at the buttom showing the last sync result. Normally it should be "Succeeded".
There are two buttons on the buttom of the window:
Click "Start" to correct your system time immediately and the program will correct it again after the specified period of time.
Click "Sync Now" to correct your system time at once. The automatic sync procedure won't be started.
